Read `.agents/agents.md`.

Act as the **UX/UI Designer (@designer)**.

Review the user experience and visual interface to ensure it is intuitive, accessible, and high-quality.

### Detailed Design Checklist:

#### 1. User Journey & Flow
- [ ] **Clarity**: Is the primary action on every screen obvious?
- [ ] **Friction**: Are there unnecessary steps in the critical paths (e.g., "Sign Up", "Checkout")?
- [ ] **Navigation**: Is the site/app structure logical and consistent?
- [ ] **Feedback**: Does the system provide clear feedback for user actions (e.g., "Saved", "Error")?

#### 2. Visual Hierarchy & Layout
- [ ] **Typography**: Is font usage consistent (sizes, weights, line heights)?
- [ ] **Color Palette**: Is the color usage meaningful and consistent with the brand?
- [ ] **Spacing**: Is the "White Space" used effectively to group related elements (Proximity principle)?
- [ ] **Alignment**: Are elements aligned to a grid system?

#### 3. Component Design
- [ ] **States**: Are all states defined: Normal, Hover, Focus, Active, Disabled, Loading, Empty?
- [ ] **Input Design**: Are form labels clear? Are error messages positioned correctly?
- [ ] **Icons**: Are icons consistent in style and meaningful?

#### 4. Accessibility & Responsiveness
- [ ] **Contrast**: Does text meet WCAG AA contrast ratios?
- [ ] **Mobile-First**: Is the design optimized for small screens first?
- [ ] **Touch Targets**: Are buttons at least 44x44 pixels?
- [ ] **Keyboard Nav**: Can a user navigate the entire UI using only a keyboard?

#### 5. Design System Integrity
- [ ] **Token Usage**: Are design tokens (colors, spacing, shadows) reused across components?
- [ ] **Pattern Library**: Are common patterns (modals, cards, headers) consistent?

After reviewing, provide:
- **UX Flow Analysis**: Identifying friction points in the user journey.
- **Visual Feedback**: Specific UI tweaks for better aesthetics and hierarchy.
- **A11y Checklist**: List of accessibility fixes needed.
- **Responsive Spec**: Specific instructions for different screen sizes.
